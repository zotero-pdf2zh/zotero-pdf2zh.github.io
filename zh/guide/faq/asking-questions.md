# 提问相关问题

关于如何有效提问的问题。

---

## 如何有效提问？🔥

### 问题描述

不知道如何正确地提问，导致问题得不到解决。

### 原因

提问信息不完整，无法让他人了解您的具体情况。

### 解决方案

### 提问前必做

1. **认真阅读文档**
   - 本项目所有的问题在所有的环境下都可以得到解决
   - 指南几乎包含了您会遇到的所有的问题
   - 请先尝试阅读终端中的错误提示，再到项目文档/常见文档中查找

2. **确定是新问题**
   - 确认您的问题不是常见问题
   - 在文档和 issue 区搜索过相关问题

### 提问时需要提供

请提供以下信息，否则可能无法得到回复：

1. **终端日志**
   - 将终端（又名：命令行/cmd）里的**所有内容**复制下来
   - 粘贴到一个 txt 文件中
   - 不要只截图部分内容，完整日志更重要

2. **Zotero 设置截图**
   - 截图您的 Zotero PDF2zh 插件设置页面
   - 确保截图清晰，可以看到所有配置

3. **错误弹窗截图**
   - 截图 Zotero 的错误弹窗（如果有）

4. **问题描述**
   - 说明您已经检查过常见问题
   - 说明您已经使用了哪些方法来解决问题
   - 说明您是否观看了某些教程

### 提问渠道

- **QQ 群**：5群 1064435415（入群答案：github）
- **GitHub Issues**：[提交问题](https://github.com/guaguastandup/zotero-pdf2zh/issues)

### 示例：好的提问

```
您好，我遇到了翻译失败的问题。

我已经查看了常见问题文档，尝试了切换端口和重启服务，但问题仍然存在。

我的环境：
- Windows 11
- Zotero 7
- Python 3.12.0
- 插件版本 3.0.37

错误信息（终端）：
[贴出完整的终端日志]

Zotero 设置：
[贴出设置截图]

Zotero 弹窗：
[贴出弹窗截图]

我已经尝试过的方法：
1. 切换端口到 9999
2. 重启了 Zotero
3. 检查了防火墙设置

请问还有什么可以尝试的方法吗？
```

### 示例：不好的提问

```
翻译失败怎么办？
```

::: tip 对比
好的提问提供了完整的环境信息和错误日志，让他人能够快速定位问题。不好的提问信息太少，无法判断问题原因。
:::

## 在群里问问题没人回复？

### 问题描述

在 QQ 群里问问题，没人回复，反而回复了别人。

### 原因

可能有以下几种情况。

### 解决方案

### 原因 1：您的问题是常见问题

**解决方案**：
- 请仔细阅读常见问题文档
- 搜索群历史消息
- 如果您认为不是常见问题，需要在问题描述时说明：**您已经查看了常见问题但无法自主解决**

### 原因 2：漏看了

**解决方案**：
- 可以过一会重新发
- 可以@群主

### 原因 3：没有发送终端信息

**解决方案**：
- 请按照"如何有效提问"的要求，提供完整的终端日志和截图
- 没有终端信息无法排查问题

## 开发者/群里的人没有给我很好的支持？

### 问题描述

认为开发者/群里的人没有给自己很好的支持，想不礼貌了。

### 原因

对免费开源项目的支持性质有误解。

### 解决方案

### 请理解以下几点

1. **本项目是免费/开源的项目**
   - 您得到的是他人用学习/工作之余的义务劳动时间
   - 插件群是为了让更多的人在此项目中受益
   - 并不能够保证有求必应

2. **请控制情绪和语言**
   - 不礼貌的言论不会解决问题
   - 反而会让他人不愿意帮助您

3. **提供者也是用户**
   - 很多回答您问题的人也是用户
   - 他们也是在用业余时间帮助您

### 如果确实需要优先支持

如果您确实需要更好的支持，可以考虑：

- **赞助项目**：支持项目开发（详见 [赞助页面](/zh/sponsors)）
- **赞助者**：如果您给作者打赏了，可以私聊联系群主，远程帮您解决问题

## 如何快速找到答案？

### 问题描述

遇到问题后，如何快速找到解决方案？

### 原因

不知道如何有效地搜索和查找信息。

### 解决方案

### 排查问题的步骤

#### 第一步：阅读终端错误信息

终端中的错误信息通常包含关键线索：

**示例 1**：
```
DeepSeek API Key is Required
```
**含义**：您没有配置自己的 DeepSeek API Key
**解决**：去配置 DeepSeek API Key

**示例 2**：
```
OSError: Microsoft Visual C++ Redistributable is not installed. Download it at https://aka.ms/vs/17/release/vc_redist.x64.exe
```
**含义**：缺少 Visual C++ 运行库
**解决**：按照链接地址下载并安装 vc_redist.x64.exe

**示例 3**：
```
Failed to canonicalize script path
```
**含义**：虚拟环境路径问题
**解决**：在项目文档/常见文档中搜索 "Failed to canonicalize"

#### 第二步：搜索项目文档

1. 在浏览器中打开本文档站点
2. 使用 Ctrl+F / Cmd+F 搜索错误关键词
3. 查看对应的解决方案

#### 第三步：搜索 GitHub Issues

1. 前往 [GitHub Issues](https://github.com/guaguastandup/zotero-pdf2zh/issues)
2. 使用搜索框搜索错误信息
3. 查看是否有类似问题和解决方案

#### 第四步：提问

如果以上方法都无法解决，按照"如何有效提问"的要求进行提问。
