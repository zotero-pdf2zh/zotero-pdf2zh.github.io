# About

Zotero PDF2zh is a plugin for translating PDFs using PDF2zh and PDF2zh_next in Zotero.

## Project Introduction

This plugin perfectly preserves original formatting, supports multiple translation engines and services, allowing you to easily translate PDF literature in Zotero.
