NetworkError when attempting to fetch resource ​

Common causes:

  1. Plugin version is too old
  2. server.py script is not running
  3. Port is occupied or firewall is blocking
  4. Antivirus software is blocking

Solution: Check plugin version, ensure server.py is running, try switching ports.
