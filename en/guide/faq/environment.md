## DLL initialization routine failed

**Cause**: Missing Visual C++ Redistributable or onnx version incompatibility.

**Solution**: Install [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) or downgrade onnx to 1.16.1.
